# To-Do List Web Application

A simple To-Do List web app built using HTML, CSS, and JavaScript.

## Features
- Add tasks
- Delete tasks
- view all tasks in a clean interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download the project
2. Open `index.html` in any browser
