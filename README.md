# To-Do List Web Application

A simple To-Do List web app built using HTML, CSS, and JavaScript.

## Features
- Add tasks
- Delete tasks
- view all tasks in a clean interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
- Copy the repository link from GitHub
- Open terminal or command prompt
- Clone the repository
- Move into the project folder
- Run the project


##Future improvements
- Add task completion feature
- Save tasks using local storage
- Edit existing tasks
- Dark mode UI
